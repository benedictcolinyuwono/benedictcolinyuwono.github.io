Autonomous Line‑Drawing Robot

Arduino · Motor control · PID · IK · 4‑wheel mobile base

Burglar Alarm with Facial Recognition

Arduino (C++) · MATLAB/Python (vision) · OOP · UART

PIN‑protected alarm system with PIR and magnetic sensors, buzzer, status LEDs, and door‑lock solenoid. Implemented modular OOP architecture on Arduino; used MATLAB/Python for face recognition and UI.
